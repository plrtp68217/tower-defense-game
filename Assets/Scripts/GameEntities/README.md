# IDamagable

То что может получать урон и умирать.

Где используется: враги, союзники, разрушаемые объекты.

# IPlacable

То что может быть размещаено на сетке.

# DamageSource

Источник урона, нужен для того что бы можно было опеределить множитель урона и тип урона.

# IEnemy 

`IDamagable` + `IPlacable` + поле `Team`
